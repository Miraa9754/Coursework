# Telegram Channel Catalog

## Основні можливості
* **Динамічна фільтрація**: Користувачі можуть миттєво фільтрувати канали за декількома категоріями (тегами) одночасно.
* **Безпечний редирект**: Всі переходи до Telegram проходять через серверний контролер (`/go`), що дозволяє приховувати прямі посилання та збирати статистику.
* **Адмін-панель**: Повний функціонал керування контентом (додавання, редагування, видалення каналів та тегів).
* **Захист даних**: Адміністративні операції захищені авторизаційним токеном `X-Admin-Token`.

## Інструкція із запуску

### 1. Сервер (Backend)
```bash
# Перейдіть у директорію сервера
cd backend

# Створіть та активуйте віртуальне середовище
python -m venv venv
# для Windows:
venv\Scripts\activate
# для macOS/Linux:
source venv/bin/activate

# Встановіть залежності
pip install -r requirements.txt

# Запустіть сервер
uvicorn main:app --reload

API буде доступне за адресою: http://127.0.0.1:8000

# Клієнт (Frontend)
# Перейдіть у директорію фронтенду
cd frontend

# Встановіть залежності (Node.js має бути встановлений)
npm install

# Запустіть додаток
npm run dev
Сайт відкриється за адресою: http://localhost:5173
